# Struct là cách để định nghĩa kiểu dữ liệu có cấu trúc, tức là gom nhiều trường dữ liệu lại với nhau.
Giống như class trong OOP, nhưng Go không có class, chỉ có struct

type Post struct {
	ID int 
	Title string
	Content string
}

Giải thích:
- Post là tên của struct, có thể hiểu là một kiểu dữ liệu mới.
- Gồm 3 thuộc tính: ID, Title, Content.

# Khởi tạo một struct có 3 cách:
1. Cách đầy đủ: 

p1 := Post{
	ID: 1,
	Title: "Hello World",
	Content: "This is my first post",
}

2. Cách rút gọn theo thứ tự:

p2:= Post{
	1, // ID
	"Hello World", // Title
	"This is my first post", // Content
}

3. Khởi tạo rỗng rồi gán sau:

var p3 Post
p3.ID = 1
p3.Title = "Hello World"
p3.Content = "This is my first post"

# In dữ liệu của struct:
fmt.Println(p1) // {1 Hello World This is my first post}
fmt.Println(p2.ID) // 1
fmt.Println("%+v\n", p3) // {ID:1
					     //  Title:Hello World 
						 //  Content:This is my first post}
